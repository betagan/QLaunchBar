About

QLaunchBar adds a small Widget to the side of your Screen which holds several, configurable Buttons to fire up your favourite applications, websites or perform other scriptable tasks. It is hidden by default and slides in once your mouse-cursor touches the edge of the screen, thus not interrupting your workflow.

Install

No binaries are released, yet. Nevertheless, you can install QLaunchBar by checking out the source code and compiling the code yourself. There is a Qt-Project file provided in the source tree. Just do

$ git clone https://github.com/betagan/QLaunchBar.git

and use QLaunchBar/QLaunchBar.pro.
